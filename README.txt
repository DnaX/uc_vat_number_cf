This module is based on and inspired by:
http://www.ubercart.org/forum/support/6856/how_extend_billing_address
It is also inspired by the VAT number module (http://www.ubercart.org/contrib/797).

This module simply adds four fields:
- Fiscal Code and Vat Number for the customer
- Fiscal Code and Vat Number for the store owner

It adds four tokens:
store-vat-number
store-cf
order-vat-number
order-cf

After you have installed this module, you have to enable the first "billing information" pane
and to disable the second "billing information" pane under admin/store/settings/checkout/edit/panes
